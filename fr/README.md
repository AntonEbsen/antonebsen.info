# Anton Meier Ebsen Jørgensen – Hub d'information (.info)

Ce projet est un hub d'information minimaliste et factuel pour Anton Meier Ebsen Jørgensen. Le site est conçu pour servir de point d'orientation neutre pour les journalistes, les hôtes de podcasts, les organisateurs de conférences et autres professionnels.

## Structure du projet

- `../index.html` : Page principale en danois.
- `../en/index.html` : Version anglaise.
- `../de/index.html` : Version allemande.
- `index.html` : Version française.
- `../style.css` : Style minimaliste partagé (Vanilla CSS).
- `../favicon.svg` : Favicon basé sur SVG.

## Principe de conception

"Cette page existe pour orienter, pas pour persuader."

## Maintenance

Pour mettre à jour le contenu :

1. Modifiez le fichier `index.html` correspondant.
2. Ajoutez de nouvelles apparitions dans les médias de manière chronologique dans la section "Apparitions publiques & médias".

## Hébergement

Le site est construit en HTML/CSS statique et peut être hébergé sur n'importe quelle plateforme.
