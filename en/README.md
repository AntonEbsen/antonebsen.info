# Anton Meier Ebsen Jørgensen – Information Hub (.info)

This project is a minimalist, factual information hub for Anton Meier Ebsen Jørgensen. The site is designed to serve as a neutral point of orientation for journalists, podcast hosts, conference organizers, and other professionals.

## Project Structure

- `../index.html`: Danish main page.
- `index.html`: English version.
- `../de/index.html`: German version.
- `../fr/index.html`: French version.
- `../style.css`: Shared minimalist styling (Vanilla CSS).
- `../favicon.svg`: SVG-based favicon.

## Design Principle

"This page exists to orient, not to persuade."

## Maintenance

To update the content:

1. Edit the relevant `index.html` file.
2. Add new media appearances chronologically under the "Public Appearances & Media" section.

## Hosting

The site is built as static HTML/CSS and can be hosted on any platform.
